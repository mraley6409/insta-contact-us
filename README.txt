== Description ==

This plugin creates a user feedback form. Users of a website with
this plugin enabled can submit their information and feedback to the
owner of the website. The plugin uses AJAX to collect the data and 
send it to the owner.

== Installation ==

1. Add the contents of the plugin folder to the '/wp-content/plugins/' directory
2. Upload the plugin through the "Plugins" menu in WordPress
3. Add a shortcode block with the label [insta-contact-us] to the page where the form will be displayed
4. Activate the plugin in the "Plugins" menu

== Frequently Asked Questions ==

1. How do i see more of the content in the feedback page?
